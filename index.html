Configurar almacenamiento

Pegá la URL del Google Apps Script que desplegaste. Tus datos se guardan en tu propio Google Sheets.

¿Cómo obtener la URL?
  1. Abrí Google Sheets → Extensions → Apps Script
  2. Pegá el contenido de Code.gs
  3. En Script Properties agregá: GEMINI_API_KEY
  4. Deploy → New deployment → Web app → Anyone → Deploy
  5. Copiá la URL y pegala arriba
